# Count vowels

Write a function `count_vowels(s)` that returns the number of vowels in the
string `s`. The vowels are `a`, `e`, `i`, `o`, `u`, counted **case-insensitively**
(so both `'a'` and `'A'` count). The letter `y` is **not** a vowel.

## Edge cases

- `count_vowels("")` returns `0`.

## Examples

```
count_vowels("hello") == 2
count_vowels("HELLO") == 2
count_vowels("xyz") == 0
count_vowels("AEIOUaeiou") == 10
count_vowels("") == 0
```
