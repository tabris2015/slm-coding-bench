# Two sum

Write a function `two_sum(nums, target)` that, given a list of integers `nums`
and an integer `target`, returns the indices `[i, j]` (with `i < j`) of the two
**distinct** elements whose values sum to `target`.

You may assume that **exactly one** valid solution exists. Return the pair of
indices as a list `[i, j]` in increasing order.

## Examples

```
two_sum([2, 7, 11, 15], 9) == [0, 1]
two_sum([3, 2, 4], 6) == [1, 2]
two_sum([3, 3], 6) == [0, 1]
two_sum([-1, -2, -3, -4, -5], -8) == [2, 4]
```
