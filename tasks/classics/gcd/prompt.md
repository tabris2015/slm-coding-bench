# Greatest common divisor

Write a function `gcd(a, b)` that returns the greatest common divisor of two
**non-negative** integers `a` and `b`.

## Edge cases

- `gcd(0, 0) == 0`
- `gcd(0, n) == n` and `gcd(n, 0) == n` for any non-negative `n`.

## Examples

```
gcd(12, 8) == 4
gcd(17, 5) == 1
gcd(0, 0) == 0
gcd(0, 7) == 7
gcd(100, 10) == 10
```
